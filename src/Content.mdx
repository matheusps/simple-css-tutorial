## Espaçamentos

Imagine dois tipos de espaçamento: interno (padding) e externo (margin).

- `padding`: espaçamento INTERNO em todos os lados (top, right, bottom, left)
- `padding-top`: espaçamento INTERNO para o topo. (análogo para os outros lados)
- `margin`: espaçamento EXTERNO em todos os lados (top, right, bottom, left)
- `margin-top`: espaçamento EXTERNO para o topo. (análogo para os outros lados)
- `...`

Exemplo:

```css
.container {
  padding: 5rem;
  background: #dddddd;
}
.item {
  padding: 2rem;
  margin: 1rem;
  background: #ff851b;
}
```

```html
<div class="container">
  <div class="item"></div>
  <div class="item"></div>
</div>
```
